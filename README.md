# cncf-stencils

Graffles stencils for CNCF projects.

All icons comes from [cncf/landscape](https://github.com/cncf/landscape/)


## Downloads

Latest artifacts in [Actions](https://github.com/alswl/cncf-stencils/actions).


## Build Required

Python 3.9+


## Build

```
make install vendor build
```
